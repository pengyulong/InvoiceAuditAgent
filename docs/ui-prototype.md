## 交互状态

### 上传状态
- **初始状态**: 拖拽区域提示文字
- **拖拽悬停**: 边框高亮，背景变色
- **上传中**: 进度条，取消按钮
- **上传完成**: 文件预览，删除选项
- **上传失败**: 错误提示，重试按钮

### 审计状态
- **等待中**: 灰色图标，待处理状态
- **处理中**: 旋转图标，进度动画
- **完成**: 绿色勾选，结果展示
- **失败**: 红色叉号，错误信息

### 响应式设计
- **桌面端** (≥1200px): 完整布局
- **平板端** (768px-1199px): 两列布局
- **手机端** (<768px): 单列布局，折叠面板

## 动画效果

### 页面切换
- **淡入淡出**: 300ms ease-in-out
- **滑动效果**: 400ms cubic-bezier(0.4, 0, 0.2, 1)

### 加载动画
- **脉冲效果**: 1.5s infinite
- **骨架屏**: 渐变加载效果
- **进度条**: 平滑进度动画

### 交互反馈
- **悬停效果**: 200ms transition
- **点击反馈**: 100ms scale效果
- **状态切换**: 300ms 颜色过渡

## Figma原型制作要点

1. **组件化设计**: 创建可复用的设计组件
2. **自动布局**: 使用Auto Layout实现响应式
3. **样式规范**: 统一的颜色、字体、间距系统
4. **交互原型**: 制作可点击的交互流程
5. **设计标注**: 添加开发所需的尺寸和样式标注
