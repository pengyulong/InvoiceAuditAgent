
### 技术架构说明

**前端架构**
- **框架**: Vue 3 Composition API + TypeScript
- **UI库**: Element Plus
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **构建工具**: Vite
- **样式**: CSS3 + Scoped CSS

**后端架构**
- **框架**: FastAPI
- **语言**: Python 3.9+
- **AI框架**: LangGraph
- **异步处理**: asyncio
- **任务队列**: Celery + Redis
- **数据库**: SQLite (开发) / PostgreSQL (生产)

**AI服务架构**
- **多模态模型**: 阿里云 DashScope (qwen-vl)
- **语言模型**: DeepSeek-V3.2-exp
- **Agent协调**: LangGraph工作流
- **图像处理**: OCR + 视觉理解

## 设计理念

### 核心设计原则
- **专业性**: 体现财务审计工具的专业性和可信度
- **简洁性**: 简化复杂流程，降低用户认知负担
- **效率性**: 优化工作流程，提升审计效率
- **可靠性**: 通过视觉设计传达系统的稳定性和准确性

### 颜色系统
- **主色调**: 深海蓝 (#1e3a8a) - 代表专业、信任、稳定
- **辅助色**: 科技蓝 (#3b82f6) - 用于交互元素和状态指示
- **成功色**: 森林绿 (#059669) - 用于通过状态和成功提示
- **警告色**: 琥珀色 (#d97706) - 用于需要人工复核的状态
- **错误色**: 深红色 (#dc2626) - 用于错误和严重问题
- **中性色**: 
  - 深灰 (#374151) - 主要文本
  - 中灰 (#6b7280) - 次要文本
  - 浅灰 (#f3f4f6) - 背景和分割线

### 字体系统
- **标题字体**: Inter (Bold/SemiBold) - 现代、清晰、专业
- **正文字体**: Inter (Regular/Medium) - 易读性强，适合长文本
- **数据字体**: JetBrains Mono - 等宽字体，用于数字和代码显示

### 视觉语言
- **卡片设计**: 使用白色卡片承载内容，提供清晰的视觉层次
- **圆角设计**: 8px圆角，营造友好而不严肃的氛围
- **阴影系统**: 微妙的阴影效果，增加界面深度
- **间距系统**: 基于8px网格的间距系统，保持一致性

## 视觉效果

### 使用的核心库
- **Anime.js**: 用于流畅的UI动画和过渡效果
- **ECharts.js**: 用于数据可视化和图表展示
- **Splitting.js**: 用于文本动画效果
- **Matter.js**: 用于Agent工作流的物理动画效果

### 动画效果
- **页面加载**: 渐入动画，从上到下的流畅展示
- **文件上传**: 拖拽时的视觉反馈，上传进度的动态展示
- **Agent处理**: 节点间的连线动画，表示数据流动
- **结果展示**: 数字的计数动画，图表的绘制动画
- **状态切换**: 平滑的颜色和尺寸过渡

### 交互效果
- **悬停效果**: 
  - 按钮: 轻微抬升 + 阴影加深
  - 卡片: 边框高亮 + 轻微缩放
  - 表格行: 背景色变化
- **点击反馈**: 按钮按下时的微动画
- **加载状态**: 骨架屏和进度指示器

### 数据可视化
- **进度环**: 使用ECharts创建动态进度环图
- **对比图表**: 清晰的柱状图和条形图
- **状态分布**: 饼图显示问题类型分布
- **时间线**: 审计过程的时间轴展示

### Hero区域效果
- **背景**: 使用shader-park创建动态几何背景
- **标题动画**: 字符逐个显现的打字机效果
- **描述文本**: 渐变色彩循环强调

### 响应式设计
- **桌面端**: 三栏布局，充分利用屏幕空间
- **平板端**: 两栏布局，保持功能完整性
- **移动端**: 单栏布局，垂直堆叠内容

### 微交互细节
- **文件拖拽**: 拖拽区域的高亮和图标变化
- **实时更新**: 数据的平滑更新动画
- **错误提示**: 温和的错误状态展示
- **成功确认**: 满足感的成功动画

## 组件设计规范

### 按钮设计
- **主要按钮**: 蓝色背景，白色文字，8px圆角
- **次要按钮**: 透明背景，蓝色边框，蓝色文字
- **危险按钮**: 红色背景，白色文字
- **最小尺寸**: 高度40px，确保触摸友好

### 表单元素
- **输入框**: 白色背景，灰色边框，聚焦时蓝色边框
- **下拉菜单**: 清晰的选项分组和悬停效果
- **文件上传**: 大尺寸的拖拽区域，清晰的图标和文字说明

### 卡片组件
- **信息密度**: 适中的内边距，避免拥挤感
- **层次结构**: 通过阴影和边框区分卡片层级
- **内容组织**: 标题、内容、操作的清晰划分
